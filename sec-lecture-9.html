    Section 2.4 Lecture 9 - Separation of variables

    Recall that the general solution to the ODE \(y'' + c^2 y = 0\) can be written in the form
    \begin{equation*} y(x) = A \cos ct + B \sin ct. \end{equation*}
    We’re going to apply that idea to certain type of PDE by a method called separation of variables.

    Subsection 2.4.1 Dirichlet conditions

    Consider the Dirichlet problem
    \begin{align*} DE:\,\, \amp u_{tt} = c^2 u_{xx}, \,\, 0 \lt x \lt \ell \\ BC:\,\, \amp u(0,t) = 0 = u(\ell,t), \,\, t \gt 0 \end{align*}
    Let us make the (strikingly strong) assumption that a solution takes the form \(u(x,t) = X(x) T(t)\text{;}\) that is, the function is independently responsive to \(x\) and \(t\text{.}\) The reason to do so is immediately apparent on calculating derivatives.
    \begin{align*} u(x,t) \amp= X(x)T(t) \\ u_{xx} \amp= X''(x)T(t) \\ u_{tt} \amp= X(x) T''(t). \end{align*}
    Plugging into the PDE then gives
    \begin{equation*} X(x)T''(t) = c^2 X(x)''T(t). \end{equation*}
    Dividing through (and introducing a convenient negative sign) gives
    \begin{equation*} -\frac{T''}{c^2 T} = -\frac{X''}{X} = \la \end{equation*}
    for some constant \(\la\text{.}\) The justification for these expressions being equal to some mutual constant \(\la\) comes from either taking partial derivatives with respect to \(x, t\) or from observing that these expressions are equal for all values of the independent variables \(x, t\text{,}\) which can only happen if they are constant. This gives rise to two separated equations.
    \begin{align*} T''(t) + \la c^2 T(t) \amp= 0 \\ X''(x) +\la X(x) \amp= 0. \end{align*}
    The solutions to the two equations will depend on the sign of the constant \(\la\text{.}\) For the moment, let us consider the case where \(\la \gt 0\) so that we can write \(\la = \beta^2\) for some positive constant \(\beta\text{.}\) In this case, the solutions to the equations are
    \begin{align*} X \amp= C \cos(\beta x) + D \sin(\beta x) \\ T \amp= A \cos(\beta c t) + B \sin(\beta c t). \end{align*}
    We now apply the initial conditions to get more information about the form of the solutions. Since \(u(0,t) = 0 = X(0)T(t)\) for all \(t\text{,}\) it must be that \(X(0) = 0\text{.}\) Hence, \(C = 0\text{.}\) Likewise, \(u(\ell, 0) = 0\) gives
    \begin{equation*} X(\ell) = D\sin(\beta \ell) = 0. \end{equation*}
    Since \(D = 0\) gives the trivial solution in \(X\text{,}\) it must be that \(\sin(\beta \ell) = 0\text{.}\) Then the possible values of \(\beta\) are \(\beta = \frac{n \pi}{\ell}\) for any positive integer \(n\text{.}\) That is, each \(n\) gives a distinct \(\beta\) and hence a distinct solution to the differential equation. Let us then say \(\beta_n = \frac{n \pi}{\ell}\) and \(\la_n = \left(\frac{n \pi}{\ell}\right)^2\text{.}\) Consequently,
    \begin{equation*} X_n = \sin\left(\frac{n \pi x}{\ell} \right) \end{equation*}
    is a solution to \(X'' + \la X = 0\) and the corresponding solution to \(T'' + \la c^2 T = 0\) is
    \begin{equation*} T_n = A_n \cos(\frac{n \pi c t}{\ell}) + B_n \sin(\frac{n \pi c t}{\ell}). \end{equation*}
    So for each \(n\text{,}\)
    \begin{equation*} u_n = (A_n \cos(\frac{n \pi c t}{\ell}) + B_n \sin(\frac{n \pi c t}{\ell}))\sin\left(\frac{n \pi x}{\ell} \right) \end{equation*}
    is asolution to the boundary value problem. As the PDE is linear, the principle of superposition tells us that
    \begin{equation*} u(x,t) = \sum_{n=1}^N (A_n \cos(\frac{n \pi c t}{\ell}) + B_n \sin(\frac{n \pi c t}{\ell}))\sin\left(\frac{n \pi x}{\ell} \right) \end{equation*}
    is also a solution.
    What about the other possibilties for \(\la\text{?}\) If \(\la = 0\text{,}\) then we are solving the problem
    \begin{equation*} X'' = 0, \end{equation*}
    which has solution \(X = ax + b\) for constants \(a,b\text{.}\) Since \(X(0) = 0 = b\text{,}\) this require that \(b = 0\text{.}\) Since \(X(\ell) = 0 = a \ell\text{,}\) this requires that \(a = 0\) and hence that the only solution is the trivial solution \(X = 0 \text{.}\) Hence, \(\la \neq 0\text{.}\) What if \(\la \lt 0\text{?}\) Then write \(\la = -\gamma^2\) for some positive constant \(\gamma\text{.}\) Then the equation we are trying to solve is \(X'' - \gamma^2 X = 0\) which has solutions of the form
    \begin{equation*} X = A \cosh \gamma x + B \sinh \gamma x. \end{equation*}
    The initial conditions require that
    \begin{equation*} X(0) = 0 = A \cosh(0) + B \sinh(0) = A, \end{equation*}
    which means that \(A = 0\text{.}\) On the other hand,
    \begin{equation*} X(\ell) = 0 = B \sinh(\gamma \ell) = 0, \end{equation*}
    which requires that \(B = 0\text{.}\) Hence \(X = 0\) is again the only soluition to the equation, and hence \(\la\) is not negative. Then solutions corresponding to the possible positive values of \(\la\) describe the entire solution space under the assumption that the solution is separated.
    Suppose that we had started the boundary value problem with an initial condition \(u(x,0) = \phi(x), u_t(x,0) = \psi(x)\text{.}\) Applying the initial condition our solution \(u\) would say that the IVP was solvable with solution \(u\) precisely when
    \begin{equation*} \phi(x) = \sum_{n=1}^N A_n \sin\left(\frac{n \pi x}{\ell} \right) \end{equation*}
    and
    \begin{equation*} \psi(x) = \sum_{n=1}^N B_n \frac{\pi n c}{\ell} \sin\left(\frac{n \pi x}{\ell} \right). \end{equation*}
    Fourier’s insight was that these observations should also hold if \(N \to \infty\) (which should at least be plausible if we could construct some sequences \(\{A_n\}, \{B_n\}\) so that the series all converge), but then we are faced with the following (rather difficult) question: for which functions \(\phi\) does it hold that one can find a sequence \(\{A_n\}\) so that \(\phi(x) = \sum_{n = 1}^\infty A_n \sin(\frac{\pi n x}{\ell})\text{?}\) This is the central question that will lead us into the study of Fourier series.
    A similar analysis on the problem
    \begin{align*} DE: \,\, \amp u_t = k u_{xx} \\ BC: \,\, \amp u(0,t) = 0 = u(\ell,t) \\ IC: \,\, \amp u(x,0) = \phi(x) \end{align*}
    leads to the solution
    \begin{equation*} u(x, t) = \sum_{n=1}^\infty A_n e^{- (\frac{\pi n}{\ell})^2 kt} \sin\left(\frac{n \pi x}{\ell} \right) \end{equation*}
    provided that a sequence \(\{A_n\}\) exists so that
    \begin{equation*} \phi(x) = \sum_{n=1}^\infty A_n \sin\left(\frac{n\pi x}{\ell}\right). \end{equation*}
    That is, the same requirement about the nature of \(\phi\) holds.

    Subsection 2.4.2 Eigenvalues and Eigenfunction.

    We can view the process we used above as linear algebraic in nature. If we rewrite the equation \(X'' + \la X = 0\) in the form
    \begin{equation*} -D^2 X = \la X, \end{equation*}
    we should be reminded of the eigenvalue/eigenvector equation \(A v = \la v\) where \(\la\) is an eigenvalue and \(v\) is an eigenvector. Certainly \(-D^2\) is a linear operator, and so we use the same terminology here: if \(X\) is a function and \(\la\) is a number so that \(-D^2 X = \la X\text{,}\) we say that \(X\) is an eigenfunction of \(-D^2\) and that \(\la\) is the corresponding eigenvalue. When we impose boundary conditions, we restrict the possible eigenvalue/eigenfunction pairs for the operator \(-D^2\text{.}\) A shorthand description of the process that we took above might be that we assumed \(u = XT\text{,}\) solved for the possible forms of the eigenfunctions, and then worked backwards to find the possible eigenvalues. We then asserted that the subspace of solutions spanned by the product of eigenfunctions in \(x\) and \(t\) solves an initial value problem if the initial condition \(\phi(x)\) is in the span of the eigenfunctions in \(x\text{.}\) For example, in the last subsection, given the wave equation with Dirichlet boundary conditions, we found that the equation \(-D^2 X = \la X\) has eigenvalues \(\la_n = \left(\frac{n\pi}{\ell}\right)^2\) with corresponding eigenfunctions \(X_n = \sin(\frac{n \pi x}{\ell})\text{.}\)

    Subsection 2.4.3 Neumann conditions

    Let us consider the case where the initial conditions are Neumann- that is, we describe the normal derivative at the boundary. In particular, let us look at the diffusion equation given by
    \begin{align*} DE: \,\, \amp u_t = k u_{xx} \\ BC: \,\, \amp u_x(0,t) = 0 = u_x(\ell,t) \\ IC: \,\, \amp u(x,0) = \phi(x) \end{align*}
    which corresponds to a heated rod of finite length with insulated ends. As before, let us assume that \(u(x,t) = X(x) T(x)\) is a separated solution. This leads to the system of equations
    \begin{align*} T'(t) + \la k T(t) \amp= 0 \\ X''(x) +\la X(x) \amp= 0. \end{align*}
    Regardless of the sign of \(\la\text{,}\) we have the solution \(T(t) = e^{-k\la t}\text{.}\) As before, we’ll use the equation in \(X\) and the boundary conditions to extract more information about the nature of the solutions.
    As before, start with the assumption that \(\la \gt 0\) so that our solutions have the form
    \begin{equation*} X = C \cos\beta x + D \sin \beta x, \end{equation*}
    and so
    \begin{equation*} X' = - C\beta \sin \beta x + D \beta \cos \beta x. \end{equation*}
    Now we are in position to apply our boundary conditions, which imply that \(X'(0) = 0 = X'(\ell)\text{.}\) First,
    \begin{equation*} X'(0) = 0 = C\beta \sin 0 + D \beta \cos 0, \end{equation*}
    so \(D = 0\text{.}\) Since
    \begin{equation*} X'(\ell) = 0 = -C\beta\sin(\beta \ell), \end{equation*}
    we must have that \(\beta = \frac{n \pi}{\ell}\) for some integer \(n\text{.}\) Then the eigenvalues of the problem are \(\la = \left(\frac{n \pi}{\ell}\right)^2\) and the corresponding eigenfunctions are \(X_n = \cos\left(\frac{n \pi x}{\ell}\right)\text{.}\)
