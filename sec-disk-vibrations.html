    Section 2.6 Vibrations on a disk

    Subsection 2.6.1 Bessel’s equation

    Subsubsection 2.6.1.1 Bessel functions

    Consider the ordinary differential equation
    \begin{equation*} x^2 y'' + x y' + (x^2 - m^2)y = 0. \end{equation*}
    This is known as Bessel’s equation of order \(m\text{.}\) Bessel’s equation has a regular singular point at \(x = 0\text{,}\) which means that we expect at least one of the solutions to be a Frobenius series:
    \begin{equation*} y_1(x) = x^m \sum_{n=0}^\infty a_n x^n. \end{equation*}
    The usual method for solving for the coefficients is to plug the series into the differential equation. Doing so here leads to the Bessel function of the first kind of order \(m\text{,}\)
    \begin{equation*} y(x) = J_m(x) = \sum_{n=0}^\infty \frac{(-1)^n}{\Gamma(n+1)\Gamma(n + m + 1)} \left(\frac{x}{2} \right)^{2n + m}. \end{equation*}
    Here, the \(\Gamma(x)\) is the Gamma function, which satisfies \(\Gamma(x+1) = x\Gamma(x)\) and generalizes the factorial function. Indeed, \(\Gamma(n+1) = n!\text{.}\) In the special case where \(m\) is a positive integer, we get the simpler
    \begin{equation*} y(x) = J_m(x) = \sum_{n=0}^\infty \frac{(-1)^n}{n! (n+m)!} \left(\frac{x}{2} \right)^{2n + m}. \end{equation*}
    Bessel functions have interesting behavior - they are oscillatory, but in a non-periodic way. Here are some plots of the first few Bessel functions \(J_m(x)\) of integer order.
    We might be interested in scaling a Bessel function so that it oscillates a fixed number of times in a given interval. This is more difficult than with a periodic function like \(\sin n x\text{,}\) where choosing a value of the positive integer \(n\) will determine the number of whole periods contained in an interval. Often, we’ll be interested in placing a zero of \(J_m\) at the end of the interval. It can be shown that each Bessel function has an infinite number of zeros; let us denote the \(n\)th zero of the Bessel function \(J_m(x)\) by \(j_{mn}\text{.}\) Every mathematical programming language has commands for computing the zeros of Bessel functions.
    Each new value of \(n\) adds another crossing to the \(x\)-axis in \([0,1]\text{.}\)
    As Bessel’s equation is second order, we expect a second linearly independent solution. If \(\alpha\) is not an integer, then the Bessel function of the second kind of order \(\alpha\) is
    \begin{equation*} Y_\alpha(x) = \frac{J_\alpha(x) \cos(\alpha x) - J_{-\alpha}(x)}{\sin(\alpha x)}. \end{equation*}
    For integer order \(m\text{,}\) we define
    \begin{equation*} Y_m(x) = \lim_{\alpha \to m} Y_\alpha(x). \end{equation*}

    An immediately obvious feature is that these functions are all unbounded at \(x=0\text{,}\) which will be worth considering when using Bessel functions to describe physical problems.

    Subsubsection 2.6.1.2 Bessel expansions

    We’re going to be using the functions \(J_m\) to model vibrations on intervals of fixed length (say \([0,1]\)). If we imagine that the vibrating medium is fixed at one end of the interval, each value of \(m\) will lead to a family of possible vibrations depending on how many "wiggles" the function takes between the origin and the fixed end at \(x=1\text{.}\) Given some initial conditions, it would be nice to be able to decompose these into a Bessel function expansion. The following theorems provide a useful way for producing such an expansion.
    Theorem 2.6.1.
    The family of functions \(\{J_m(j_{mn} x)\}_{n=0}^\infty\) is an orthogonal basis for the weighted L^2 space on \([0,1]\) with weighted inner product \(\langle f, g \rangle = \int_0^1 f(x) g(x) \, xdx\text{.}\)
    For convenience, we’ll denoted this weighted \(L^2\) space by \(L^2([0,1], xdx)\text{.}\)
    Theorem 2.6.2. Fourier-Bessel Expansion.
    Suppose that \(f \in L^2([0,1], xdx)\text{.}\) Then \(f\) has a Bessel function expansion
    \begin{equation*} f(x) = \sum_{n=0}^\infty c_n J_m(j_{mn}x) \end{equation*}
    where
    \begin{equation*} c_n = \frac{\langle f(x), J_m(j_{mn} x )\rangle}{\norm{ J_m(j_{mn} x)}^2}. \end{equation*}
    One way to think about this type of expansion is that we’re decomposing \(f\) into a series of vibrations of overall mode \(m\) that build it up (very much like periodic functions being approximated by sines and cosines). Note that for each \(m\text{,}\) we get another basis.

    Subsection 2.6.2 Radial Vibrations

    Let us consider vibrations of a drumhead attached to a circular hoop. Suppose the membrane is attached to the hoop, which has radius \(1\) (by rescaling, we can easily adjust this to have any radius we choose). As we’re working in a circular domain, we’ll use polar coordinates, so that every point in the disk \(D\) can be written \((r, \theta)\) where \(0 \lt r \lt 1\) and \(-\pi \lt \theta \lt \pi\text{.}\)
    We’re going to need a polar version of the wave equation, which in turn requires a polar version of the Laplace operator \(\Delta u = u_{xx} + u_{yy}.\) It turns out that the standard polar coordinate change, the chain rule, and a lot of elbow grease will be us to
    \begin{equation*} \Delta u = u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\theta \theta}. \end{equation*}
    In this setting, the wave equation becomes
    \begin{equation*} u_{tt} =c^2 \Delta u = c^2(u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\theta \theta}). \end{equation*}
    Then the system we’re interested in is
    \begin{align*} DE: \,\, \amp u_{tt} =c^2 \Delta u = c^2(u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\theta theta}) \\ BC: \,\, \amp u(1, \theta, t) = 0 \\ IC: \,\, \amp u(r,\theta, 0) = f(r, \theta) \\ \amp u_t(r, \theta, 0) = g(r, \theta). \end{align*}
    Now we’ll proceed by separation of variables. Let \(u(r, \theta, t) = R(r) \Theta(\theta) T(t)\text{.}\) Under this substitution, the equation becomes
    \begin{equation*} T'' R\Theta = c^2(R'' \Theta T + \frac{1}{r} R' \Theta T + \frac{1}{r^2} R \Theta'' T) \end{equation*}
    which after clearing denominators is
    \begin{equation*} \frac{T''}{c^2 T} = \frac{R''}{R} + \frac{1}{r} \frac{R'}{R} + \frac{1}{r^2} \frac{\Theta''}{\Theta} = -\lambda. \end{equation*}
    As usual, we set equal functions of independent variables equal to a constant. We get our first equation here,
    \begin{equation*} T'' + c^2 \la T = 0, \end{equation*}
    and we expect the solutions to oscillate from the physical problem, which gives \(>\la \gt 0\) and
    \begin{equation*} T(t) = e \cos(c\sqrt{\la}t) + f\sin(c\sqrt{\la}t). \end{equation*}
    To separate the other variables, first write
    \begin{equation*} \frac{R''}{R} + \frac{1}{r} \frac{R'}{R} +\la = -\frac{1}{r^2}\frac{\Theta''}{\Theta} \end{equation*}
    then finish the separation step by multiplying through by \(-r^2\) to get
    \begin{equation*} -r^2\frac{R''}{R} - r \frac{R'}{R} -\la r^2 = \frac{\Theta''}{\Theta} = - \mu. \end{equation*}
    Our second equation is
    \begin{equation*} \Theta'' + \mu \Theta = 0 \end{equation*}
    which has the hidden boundary condition \(\Theta(-\pi) = \Theta(\pi)\) and \(\Theta'(-\pi) = \Theta'(\pi)\text{,}\) conditions that hold as the drum membrane should be continuous and smooth as we pass through the negative \(x\)-axis. These are periodic boundary conditions, and imposing them leads to the conculusion that \(\mu = m^2\) for non-negative integers \(m\) and that for each such \(m\) we have
    \begin{equation*} \Theta_m = a_m \cos(m\theta) + b_m \sin(m \theta). \end{equation*}
    Now let us turn to the third equation, which is
    \begin{equation*} r^2\frac{R''}{R} + r \frac{R'}{R} +\la r^2 - \mu = 0 \end{equation*}
    or after clearing out the \(R\) in the denominator and substituting in for \(\mu\text{,}\)
    \begin{equation*} r^2 R'' + r R' + (\la r^2 - m^2) R = 0. \end{equation*}
    Evidently, this is a form of Bessel’s equation. We can state the exact form we described above by making the change of variables \(\rho = \sqrt{\la} r\text{,}\) to get
    \begin{equation*} \rho^2 R''(\rho) + \rho R'(\rho) + (\rho^2 - m^2) R(\rho) = 0, \end{equation*}
    which is precisely the Bessel equation of integer order \(m\text{.}\) One form of the general solution to this equation is a linear combination of Bessel functions,
    \begin{equation*} R_m(\rho) = k_m J_m(\rho) + d_m Y_m(\rho). \end{equation*}
    Since the functions \(Y_m\) are unbounded, and we are considering a drum membrane, it should be that \(\lim_{\rho \to 0} \abs{R_m(\rho)} \lt \infty\text{,}\) which implies that \(D_m = 0\) for all \(m\text{.}\) Our other boundary condition is that \(R_m(1) = 0\text{,}\) which was in the \(r\) variable. Since \(\rho = \sqrt{\la}r\text{,}\) we have \(r=1 \Rightarrow \rho = \sqrt{\la}\text{.}\) This means that
    \begin{equation*} R_m(\sqrt{\la}) = k_m J_m(\sqrt{\la}) = 0. \end{equation*}
    This equation can only be true when \(\sqrt{\la} = j_{mn}\) for some zero of \(J_m\text{.}\) That is, we know that for each \(m\text{,}\) it must be that \(\la = j_{mn}^2\) where \(n = 0, 1, 2, \ldots\text{.}\) Hence, for each \(m\text{,}\)
    \begin{equation*} R_m(r) = \sum_{n=0}^\infty k_{mn} J_m(j_{mn} r). \end{equation*}
    We can now also definitively state the form of \(T\) as well:
    \begin{equation*} T_{mn}(t) = e_{mn} \cos(cj_{mn}t) + f_{mn}\sin(cj_{mn}t). \end{equation*}
    We can now state the full form of our solution \(u(r, \theta, t)\text{.}\)
    \begin{equation*} u(r, \theta, t) = \sum_{m=0}^\infty \sum_{n=0}^\infty J_m(j_{mn} r)(a_m \cos(m\theta) + b_m \sin(m \theta))(e_{mn} \cos(cj_{mn}t) + f_{mn}\sin(cj_{mn}t)) \end{equation*}
    This looks admittedly daunting. It easier to understand if we think of just the spatial functions \(J_m(j_{mn} r)(a_m \cos(m\theta) + b_m \sin(m \theta))\text{,}\) each of which represents a mode of vibration determined entirely by \(m, n\text{.}\) Because of the periodicity of \((a_m \cos(m\theta) + b_m \sin(m \theta))\text{,}\) each value of \(m\) corresponds to a fixed number of "nodal lines", which will have constant value \(0\text{.}\) Likewise, the integer \(n\) represents the number of "turns" the Bessel function takes in the interval \([0,1]\text{,}\) which when swept out in a circle create "nodal circles".
