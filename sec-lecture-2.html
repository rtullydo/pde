    Section 1.2 Lecture 2

    Subsection 1.2.1 First Order Linear Equations

    Consider the equation
    \begin{equation*} a u_x +b u_y = 0, \end{equation*}
    where \(a, b\) are constants, not both zero. We’ll being by looking at two initial methods for solving this equation. First, we’ll introduce what is sometimes called the method of characteristics. We want to find curves in \(\R^2\) on which the PDE acts like an ODE (which will allow us to use basic ODE techniques in the solution). First, observe that
    \begin{equation*} a u_x + b u_y = (a,b) \cdot \nabla u = \norm{(a,b)} D_{(a,b)} u, \end{equation*}
    the directional derivative of \(u\) in the direction \((a,b)\text{.}\) Where is such a \(u\) constant? The value of \(u\) is constant when the directional derivative is zero, which happens on any line parallel to \((a,b)\text{,}\) all of which have the form \(bx - ay = k\) for constants \(k\text{.}\) (These are called the characteristic lines.) But then the value of \(u\) at \((x,y)\) only depends on which particular line \((x,y)\) happens to fall on, which is to say \(u\) only depends on the value \(b x - ay\text{.}\) Another way to state this is that the solution to \(a u_x + b u_y = 0\) must have the form \(u(x,y) = f(b x - a y)\) where \(f\) is any differentiable function of one variable. For example, we can show that when \(f = x^2\text{,}\) the function \(u(x,y) = (b y - ax)^2\) is a solution.
    \begin{align*} u(x,y) \amp = (bx - ay)^2 \\ u_x(x,y) \amp = 2 (bx - ay) b \\ u_y(x, y) \amp = 2 (bx - ay) (-a) \end{align*}
    and so
    \begin{equation*} a u_x + b u_y = a( 2 (bx - ay) b) + b(2 (bx - ay) (-a)) = 0. \end{equation*}
    This also suggests a linear change of variables. Let \(X = a x + b y\) and \(Y = bx - ay\) (this is essentially decomposing the function via the gradient at each point).
    \begin{align*} u_x \amp = \frac{\partial u}{\partial x} = \frac{\partial u}{\partial X} \frac{\partial X}{\partial x} + \frac{\partial u}{\partial Y} \frac{\partial Y}{\partial x} = a u_X + b u_Y \\ u_y \amp = \frac{\partial u}{\partial y} = \frac{\partial u}{\partial X} \frac{\partial X}{\partial y} + \frac{\partial u}{\partial Y} \frac{\partial Y}{\partial y} = b u_X -a u_Y \end{align*}
    Then \(a u_x + b u_y = 0\) becomes \((a^2 + b^2) u_X = 0\text{,}\) which is an ODE in \(X\text{.}\) Since \((a^2 + b^2) \neq 0\text{,}\) solving \(u_X = 0\) gives \(u = f(Y)\) for some \(C^1\) function \(f\text{,}\) which is to say that \(u(x, y) = f(Y) = f( bx - ay)\text{.}\)

    Example 1.2.1. A linear coefficient equation.

    Solve \(4 u_x - 3 u_y = 0\) given the intial condition \(u(0, y) = y^3\text{.}\)
    By the above discussion, we know that the solution has the form \(u(x,y) = f(-3x -4y)\text{.}\) We have enough information to find \(f\text{.}\) Since \(y^3 = u(0,y) = f(-4 y)\text{,}\) set \(w = -4y\text{.}\) Then \(f(w) = -\frac{1}{64} w^3\text{.}\) Hence, the solution to the PDE with initial condition is \(u(x,y) = -\frac{1}{64}(-3 -4y)^3\text{.}\)

    Example 1.2.2. A variable coefficient equation.

    Consider the equation
    \begin{equation*} u_x + y u_y = 0. \end{equation*}
    As before, notice that the equation implies that
    \begin{equation*} (1, y) \cdot (u_x, u_y) = 0, \end{equation*}
    which means that curves with tangent vector given by \((1,y)\) keep \(u\) constant. A curve with tangent vector given by \((1,y)\) has slope \(y\) at every point, and so the curve satisifies the ODE
    \begin{equation*} \frac{dy}{dx} = y. \end{equation*}
    Solutions to this equation are functions of the form \(y = C e^x.\) (These are the characteristic curves.) On each such curve, \(u\) is constant, which we can check with the chain rule:
    \begin{equation*} \frac{d}{dx} u(x, C e^x) = \frac{\partial u}{\partial x} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial x} = u_x + y u_y = 0. \end{equation*}
    This means that the value of \(u\) on the curve \(y = C e^x\) is determined entirely by \(C\text{,}\) as \(u(x, Ce^x) = u(0, C)\text{.}\)
    As \(y = C e^x\text{,}\) we have \(C = e^{-x} y\text{,}\) and so
    \begin{equation*} u(x, Ce^x) = u(0, C) = u(0, e^{-x}y). \end{equation*}
    That is, \(u\) is a function of \(e^{-x}y\text{.}\) Thus, \(u(x, y) = f(e^{-x} y)\) is the general solution, where \(f\) is an arbitrary \(C^1\) function.

    Example 1.2.3. A more complicated example.

    Consider the equation
    \begin{equation*} u_x + 2x y^2 u_y = 0. \end{equation*}
    We can rewrite this as \((1, 2xy^2) \cdot (u_x, u_y) = 0\text{,}\) and so \(u\) is constant along curves with tangent vectors given by \((1, 2xy^2)\text{.}\) This leads to the ODE
    \begin{equation*} \frac{dy}{dx} = 2x y^2 \end{equation*}
    which has solution implictly given by
    \begin{equation*} -\frac{1}{y} = x^2 + C. \end{equation*}
    and so \(u\) depends on \(C = x^2 + \frac{1}{y}.\) Then the general solution is \(u(x,y) = f(x^2 + \frac{1}{y})\) where \(f\) is arbitrary.
