    Section 2.7 Harmonic functions on the disk

    Subsection 2.7.1 The Dirichlet problem

    As Strauss notes, the situation on rectangles seems quite similar to what we’ve seen before in the heat and wave cases. The separation of variables argument leads to very similar sorts of solutions. Things are quite a bit more interesting on the disk. Let us consider the Dirichlet problem on the disk,
    \begin{align*} DE: \,\, \amp \Delta u = 0, \,\, x^2 + y^2 \lt a^2 \\ IC: \,\, \amp u = h(\theta), \,\, x^2 + y^2 = a^2. \end{align*}
    Our usual separation of variables \(u = R(r) \Theta(\theta)\) leads to the equation
    \begin{equation*} \Theta'' + \lambda \Theta = 0, \end{equation*}
    with the requirement of the periodic boundary condtion \(\Theta(\theta) = \Theta(\theta + 2\pi)\text{.}\) Applying the boundary condition gives \(\la = n^2\) and
    \begin{equation*} \Theta_n(\theta) = A \cos n \theta + B \sin n \theta, \,\, n = 1, 2, 3, \ldots, \end{equation*}
    as well as
    \begin{equation*} \Theta_0(\theta) = A \end{equation*}
    when \(\la = 0\text{.}\)
    Moving on to the equation \(r^2 R'' + rR' - \la R = 0\text{,}\) which becomes
    \begin{equation*} r^2 R'' + rR' - n^2 R = 0, \end{equation*}
    we recognize a Cauchy-Euler equation, which has solutions of the form \(R = r^\alpha\text{.}\) Plugging in,
    \begin{align*} \alpha(\alpha - 1)r^\alpha + \alpha r^\alpha - n^2 r^\alpha \amp= 0 \\ (\alpha(\alpha - 1)+ \alpha - n^2 )r^\alpha \amp= 0 \\ (\alpha^2 - n^2 )r^\alpha \amp= 0. \end{align*}
    Since \(r^\alpha \neq 0\text{,}\) we get \(\alpha = \pm n\text{,}\) and hence that
    \begin{equation*} R(r) = Cr^n + Dr^{-n}, \,\, n = 1, 2,3, \ldots, \end{equation*}
    and in the case of \(n=0\text{,}\) we have \(R = C\) for some constant \(C\text{.}\) We’ll need a second linearly independent solution in this case, which is provided by \(\log r\) (say by variation of parameters), so that
    \begin{equation*} R_0(r) = C + D \log r. \end{equation*}
    We now have a family of separated solutions,
    \begin{equation*} u_n = (Cr^n + Dr^{-n})(A \cos n\theta + B \sin n \theta), \,\, n = 1, 2, 3, \ldots \end{equation*}
    and
    \begin{equation*} u_0 = (C + D\log r)(A). \end{equation*}
    The issue is that some of these solutions are unbounded at \(r = 0\text{.}\) As we see twice differentiable functions on the whole disk of radius \(a\text{,}\) we need to impose a “hidden” boundary condition \(\lim_{r \to 0} u \lt \infty\text{.}\) This means that we must dispense with the logarithm and the rational parts of the full set of harmonic solutions we derived above. We conclude that our separated solution has the form
    \begin{equation*} u(r, \theta) = \frac{1}{2}A_0 + \sum_{n=1}^\infty r^n (A_n \cos n\theta + B_n \sin n \theta). \end{equation*}
    Finally, we apply the inhomogeneous boundary condition. Setting \(r = a\text{,}\) we get
    \begin{align*} h(\theta) \amp= \frac{1}{2}A_0 + \sum_{n=1}^\infty a^n (A_n \cos n\theta + B_n \sin n \theta)\\ \amp= \frac{1}{2}A_0 + \sum_{n=1}^\infty (a^n A_n \cos n\theta + a^n B_n \sin n \theta). \end{align*}
    That is, this is a full Fourier series for \(h(\theta)\text{,}\) and so our coefficient formulas tell us that
    \begin{equation*} A_n = \frac{1}{\pi a^n} \int_0^{2\pi} h(\phi) \cos(n \phi) \, d\phi \end{equation*}
    and
    \begin{equation*} B_n = \frac{1}{\pi a^n} \int_0^{2\pi} h(\phi) \sin(n \phi) \, d\phi \end{equation*}
    We already have an expression then for the solution to the Dirichlet problem on the disk, but now something rather magical happens. It turns out that we can actually sum up the series expansion to get a rather nice formula for the solution. Plugging our expressions for the coefficients into the series and combining integrals, we get
    \begin{align*} u(r,\theta) \amp= \frac{1}{2\pi} \int_0^{2\pi} h(\phi) \, d\phi + \sum_{n=1}^\infty \frac{r^n}{\pi a^n}\int_0^{2 \pi} h(\phi)(\cos n\phi \cos n \theta + \sin n \phi \sin n \theta)\, d\phi \\ \amp= \frac{1}{2\pi}\int_0^{2\pi} h(\phi) \left[1 + 2 \sum_{n=1}^\infty \left(\frac{r}{a}\right)^n \cos n (\theta - \phi) \right]\, d \phi \end{align*}
    Now let’s focus on the term in the brackets and flip over to complex numbers.
    \begin{align*} \amp \left[1 + 2 \sum_{n=1}^\infty \left(\frac{r}{a}\right)^n \cos n (\theta - \phi) \right] \\ \amp= 1 + \sum_{n=1}^\infty (\frac{r}{a})^n \left(\frac{e^{in(\theta - \phi)}+ e^{-in(\theta - \phi)}}{2}\right) \\ \amp= 1 + \sum_{n=1}^\infty (\frac{r}{a})^n e^{in(\theta - \phi)} + + \sum_{n=1}^\infty (\frac{r}{a})^n e^{-in(\theta - \phi)} \\ \amp= 1 + \frac{\frac{r}{a} e^{i(\theta - \phi)}}{1 - \frac{r}{a} e^{i(\theta - \phi)}} + \frac{\frac{r}{a} e^{-i(\theta - \phi)}}{1 - \frac{r}{a} e^{-i(\theta - \phi)}} \\ \amp= \frac{a^2 - r^2}{a^2 - 2 ar\cos(\theta - \phi) + r^2} \end{align*}
    where we’ve used the geometric series formula in the move from the third to the fourth line.

    Subsection 2.7.2 The Poisson integral

    This object is so important, we give it both a name and a symbol. The Poisson kernel \(P_r(\alpha)\) is the function
    \begin{equation*} P_r(\alpha) = \frac{a^2 - r^2}{a^2 - 2 ar\cos\alpha + r^2} \end{equation*}
    Plugging this back into our expression for \(u\text{,}\) we get
    \begin{align*} u(r,\theta) \amp= \frac{1}{2\pi}\int_0^{2\pi} h(\phi) \left[1 + 2 \sum_{n=1}^\infty \left(\frac{r}{a}\right)^n \cos n (\theta - \phi) \right]\, d \phi \\ \amp= \frac{1}{2\pi} \int_0^{2 \pi} h(\phi) P_r(\theta - \phi)\, d\phi, \end{align*}
    which is often called the Poisson integral. The function \(P_r(\theta)\) plays a similar role for the Dirichlet problem on the disk that the heat kernel did in the heat equation on the line.
    Before we go on, it’s worth noting how strong this result actually is - we’re saying that given a function on a circle \(h(\theta)\text{,}\) we can recover the exact harmonic function on the inside of the disk that limits to that circle on the boundary. This is like saying that one can recover the function \(f(x)\) on an interval \((a, b)\) by only knowing the values of the function at \(a\) and \(b\text{!.}\) In fact, the uniqueness of solutions to the Dirichlet problem arising from the maximum principle gives us the following theorem.

    Theorem 2.7.1.

    Let \(h(\phi)\) be a continuous function on the circle \(C = \mathrm{bdy} D\text{.}\) Then
    \begin{equation*} u(r, \theta) = \frac{1}{2\pi} \int_0^{2 \pi} h(\phi) P_r(\theta - \phi)\, d\phi \end{equation*}
    is the only harmonic function on \(D\) with \(\lim_{r \to 1}u(r, \theta) = h(\theta)\) for all \(\theta\text{.}\)
    The Poisson kernel has all kinds of nice properties, as well as some profound consequences for the behavior of hamronic functions. Our first property is a sort of Fourier expansion for the Poisson kernel.

    Proposition 2.7.2.

    Let \(D\) be the unit disk \(x^2 + y^2 \lt 1\text{.}\) Then
    \begin{equation*} P_r(\alpha) = 1 + 2 \sum_{k=1}^\infty r^k \cos k \alpha. \end{equation*}

    Proof.

    Recall that since we’re on the unit disk that \(a = 1\text{.}\) Then
    \begin{equation*} P_r(\alpha) = \frac{1 - r^2}{1 - 2r\cos\alpha + r^2}. \end{equation*}
    Then
    \begin{equation*} 1- 2r \cos\alpha + r^2 = 1 - r(e^{i\alpha} + e^{-i\alpha}) + r^2 = (1- re^{i\alpha})(1-re^{-i\alpha}). \end{equation*}
    Then
    \begin{align*} P_r(\alpha) \amp= \frac{1 - r^2}{1 - 2r\cos\alpha + r^2}\\ \amp= \frac{1 - r^2}{(1- re^{i\alpha})(1-re^{-i\alpha})}\\ \amp = (1-r^2) \frac{1}{1 - re^{i\alpha}} \frac{1}{1 - re^{-i\alpha}}.\\ \amp= (1-r^2) \sum_{n=0}^\infty r^n e^{in\alpha} \sum_{m=0}^\infty r^m e^{-im\alpha} \\ \amp= (1-r^2) \sum_{n=0}^\infty \sum_{m=0}^\infty r^{n+m} e^{i(n-m)\alpha} \\ \amp= (1-r^2) \sum_{k=-\infty}^\infty \frac{1}{1-r^2} r^\abs{k} e^{ik\alpha} \\ \amp= \sum_{k=-\infty}^\infty r^\abs{k} e^{ik\alpha} \\ \amp = 1 + 2 \sum_{k=1}^\infty r^k \cos k \alpha. \end{align*}
    Proposition 2.7.2 gives a nice way to compute some easy examples. Consider the Dirichlet problem on the unit disk where we require that \(h(\theta) = \cos(n \theta)\text{.}\) This is a continuous function on the unit circle and so should have a unique harmonic extension to the disk. Then
    \begin{align*} u(r, \theta) \amp= \frac{1}{2\pi} \int_0^{2\pi} \cos(n \phi) P_r(\theta - \phi)\, d\phi \\ \amp=\frac{1}{2\pi} \int_0^{2\pi} \cos(n\phi) (1 + 2 \sum_{k=1}^\infty r^k \cos k(\theta - \phi))\, d\phi\\ \amp=\frac{1}{2\pi} \int_0^{2\pi} \underbrace{\cos(n\phi) (1 + 2 \sum_{k=1}^\infty r^k(\cos k \phi \cos k \theta + \sin k \phi \sin k \theta))}_{\text{orthogonality kills all but } k = n}\, d\phi\\ \amp= \frac{1}{\pi} \int_0^{2\pi} r^n \cos^2 n\phi \cos n \theta \\ \amp= r^n \cos n \theta \end{align*}
    Here’s a picture of the solution in the case where we require the harmonic extension of \(h(\theta) = \cos(2\theta)\) on the unit circle.
    We might as well check that our proposed solution does indeed meet the requirements. It isn’t hard to see that \(\lim_{r \to 1} r^n \cos n \theta = \cos n \theta\text{,}\) so the function has the correct limiting behavior. To see that it is indeed harmonic, we can check the polar Laplacian.
    \begin{align*} \Delta (r^n \cos n\theta) \amp = n(n-1) r^{n-2} \cos n \theta + \frac{1}{r} n r^{n-1} \cos n \theta + \frac{1}{r^2} (-n^2) \cos n \theta \\ \amp= r^{n-2}(n(n-1) + n - n^2)\cos n \theta \\ \amp= 0. \end{align*}
    We conclude that we have found the correct harmonic extension.
    The Poisson integral leads to some quite strict conclusions about the behavior of harmonic functions. Consider what happens if we evaluate \(u\) at \(r = 0 \text{.}\)
    \begin{equation*} u(0, \theta) = \frac{1}{2\pi} \int_0^{2\pi} h(\phi) \frac{1 - 0}{1 - 0 + 0}\, d\phi = \frac{1}{2\pi} \int_0^{2\pi} h(\phi)\, d\phi. \end{equation*}
    That is, the value of \(u\) at the center of the unit disk is precisely the average value of \(h\) around the unit circle. This quite striking property turns out to be true in general; the value of a harmonic function at a point in the domain is precisely the average value of the function on a circle centered at that point. This is called the (Gauss) mean value theorem.

    Theorem 2.7.4. Mean value theorem for harmonic functions.

    Let \(u\) be harmonic on a disk \(D\) and continuous on \(\cc{D}\text{.}\) Then the the value of \(u\) at the center of \(D\) is the average value of \(u\) on \(\mathrm{bdy} D\text{.}\)
    Harmonic functions are also strikingly smooth as a consequence of the structure of the Poisson kernel. Essentially, as the Poisson kernel is differentiable to all orders, and \(u\) is a convolution of a continuous function with a smooth function, \(u\) will also be very nicely behaved.

    Theorem 2.7.5.

    Let \(u\) be a harmonic function on an open set \(D\) in the plane. Then \(u\) has all partial derivatives of all orders on \(D\text{.}\)
    Harmonic functions actually provide the structural core of the analytic funtions of complex analysis. The nice properties of harmonic functions are inherited by the analytic functions that they comprise in that setting.
